About Desktop Share

Usage

Desktop share is powered by Node.js and Angular and is open source. It transmits data within your local network and does not interact with the cloud nor does your data leave your network.

Since there is no signing certificate, your data is trasmitted locally using http. Make sure there is no one intercepting your local traffic.
